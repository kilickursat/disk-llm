Getting Started

Installation

Disk-LLM is an inspectable disk-backed LLM research kit. Install the base package or include optional extras for Hugging Face integration, benchmarking, Gradio demos, and testing.

Terminal pip install -e .

Optional extras:

Terminal pip install -e .[hf,demo,test,bench]

Requirement Note

Requires Python ≥ 3.11. For HF parity or CPU-baseline benchmarks, ensure a CPU PyTorch build is available in the environment.

02

Model Conversion

Disk-LLM re-organizes weights into layer-oriented shards. This alignment ensures that when a transformer block requests its weights, the OS can fetch exactly those bytes from disk in a single sequential I/O operation.

The default v1 converter targets the text-only path: embed_tokens, model.layers.*, model.norm, and lm_head. Known multimodal tensors like visual.* are skipped.

Inspect
disk-llm inspect --source-dir /path/to/Qwen3.5-9B
Convert
disk-llm convert /path/to/Qwen3.5-9B ./packed-qwen35
Verify Manifest
disk-llm inspect --manifest ./packed-qwen35/manifest.json

Packed Shard Structure

Shard Contents Format
embeddings/embeddings.bin model.embed_tokens memmap
layers/layer_000.bin model.layers.0.* memmap
layers/layer_001.bin model.layers.1.* memmap
final/final.bin model.norm + lm_head memmap

04

Benchmarks

Run repeated benchmark cases across prompt lengths with Disk-LLM vs HF CPU baseline comparison.

Run benchmarks
python scripts/benchmark.py ./packed-qwen35/manifest.json \
--prompt "Explain disk-backed inference in one paragraph." \
--tokenizer /path/to/Qwen3.5-9B \
--backends disk_llm,hf_cpu \
--prompt-lengths 8,64,256,512 \
--runs 3 --output-dir ./benchmark-results
Generate plots
python scripts/plot_results.py ./benchmark-results

Outputs: benchmark_runs.csv, benchmark_summary.csv, memory_timeline.csv, throughput charts, latency curves, and RSS timeline plots.

05

Modal Remote Run

Run the entire Qwen benchmark workflow on Modal cloud infrastructure. The 9B model stays in Modal's volume — never touches your local machine.

Windows PowerShell
powershell -ExecutionPolicy Bypass -File .\scripts\run_modal_qwen35_9b.ps1
Linux / macOS
bash scripts/run_modal_qwen35_9b.sh
Pull results back
modal volume get disk-llm-benchmarks /vol/results/Qwen--Qwen3.5-9B/qwen35-9b-modal-cpu ./modal-results
